# Adobe Business Process Analysis (ADBE)

This project analyzes the business performance of Adobe Inc. using data science techniques.

Stock data was collected using Python and the yfinance library.  
A Linear Regression model was created using scikit-learn to predict trends.  
A Power BI dashboard was used to visualize stock prices and key performance indicators.
